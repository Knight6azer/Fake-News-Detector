How It Works

A two-stage pipeline combining classical NLP with deep learning.

01
✂️

Text Preprocessing

Raw text is cleaned, lowercased, stopwords removed, and words are lemmatized using NLTK to extract meaningful tokens.

→
02
📊

Feature Extraction

TF-IDF (up to 50K n-gram features) captures word importance patterns. Title is weighted 3× since fake news is tone-heavy in headlines.

→
03
🧠

AI Classification

A Logistic Regression or BiLSTM neural network classifies the article. Ensemble mode blends both for maximum accuracy.

→
04
💡

Explainability

Word importance scores highlight which words pushed the verdict. Red = fake indicator. Green = real indicator.

⚡

TF-IDF + Logistic Regression

Fast (<50ms) ~95% accuracy Explainable

Classical NLP approach using Term Frequency–Inverse Document Frequency with n-gram features (1 to 3-grams) and regularized logistic regression. Interpretable via feature coefficients.

⭐ Recommended
🧠

BiLSTM + Conv1D

Deep Learning ~97% accuracy Context-aware

Bidirectional LSTM processes word sequences in both directions, capturing long-range dependencies. Conv1D layer detects local patterns before the BiLSTM layer.

🔮

Ensemble Model

Best of both Weighted avg Most robust

Weighted combination: 40% TF-IDF + 60% BiLSTM. Reduces individual model errors and provides the most reliable detection across diverse news styles.
